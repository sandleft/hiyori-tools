## 📖 簡介

一款輕量高效的瀏覽器擴充功能，幫助你批次管理網址、一鍵收割網頁內容。適合日常收藏、資料歸檔、知識管理等場景。

---

## ✨ 主要功能

### 🔗 網址管理
- 批次保存網址，支援一鍵打開所有分頁
- 一鍵複製目前視窗所有網址
- 排除清單：自動跳過指定網址

### 📥 網頁收割
- 截圖保存（`.png`）
- 文件保存（`.md`）
- PDF 保存（呼叫瀏覽器列印）
- 整張網頁截圖（長頁面自動拼接）
- 同步到 Notion（可自訂欄位名稱）

### ⚡ 自動化
- 打開網頁後自動保存，可設定延遲時間

### 🎀 特點
- 介面簡潔，操作直觀
- 本地運行，資料不上傳
- 支援 Notion API 整合
- 自動排除不想要的網址

## 🚀 使用教學

### 1. 安裝

從 [Chrome 線上應用程式商店](https://chromewebstore.google.com) 或 [Microsoft Edge 附加元件](https://microsoftedge.microsoft.com/addons) 安裝 **Hiyori Tools**。

安裝後，點擊瀏覽器工具列的拼圖圖示，將 Hiyori Tools **釘選**起來，方便隨時存取。

---

### 2. 網址管理

#### 批次保存與開啟網址
1. 點擊 Hiyori Tools 圖示開啟面板。
2. 在文字區域貼上網址清單（每行一個網址）。
3. 點擊 **「全部開啟」** — 所有網址立即在新分頁中開啟。

#### 一鍵複製目前視窗所有網址
1. 開啟面板。
2. 點擊 **「複製全部網址」** — 目前視窗的所有網址會複製到剪貼簿。

#### 排除清單
1. 前往 **設定** → **排除清單**。
2. 新增你想自動跳過的網址或網域規則。
3. 批次操作時，這些網址將被忽略。

---

### 3. 網頁收割

前往你想保存的頁面，點擊 Hiyori Tools 圖示。

| 格式 | 操作方式 |
|------|---------|
| 📸 截圖（`.png`） | 點擊 **「截圖保存」** — 儲存目前可見區域的截圖 |
| 📄 文件（`.md`） | 點擊 **「Markdown 保存」** — 將頁面內容提取為 `.md` 檔案 |
| 🖨️ PDF | 點擊 **「PDF 保存」** — 開啟瀏覽器列印對話框（選擇「儲存為 PDF」） |
| 📜 整頁截圖 | 點擊 **「整頁截圖」** — 擷取並自動拼接整個可捲動頁面 |
| 🗂️ 同步到 Notion | 請見下方 Notion 設定 |

---

### 4. Notion 整合設定

1. 前往 [https://www.notion.so/my-integrations](https://www.notion.so/my-integrations) 建立新的整合。
2. 複製 **內部整合金鑰（Token）**。
3. 開啟目標 Notion 資料庫 → 點擊 **「...」** → **「連線」** → 連接你的整合。
4. 從資料庫網址（最後一個 `/` 和 `?` 之間的字串）複製 **資料庫 ID**。
5. 在 Hiyori Tools 中，前往 **設定** → **Notion**：
   - 貼上 **API Token**
   - 貼上 **資料庫 ID**
   - 視需要自訂欄位名稱
6. 在任意網頁上，點擊 **「同步到 Notion」** 即可儲存至你的資料庫。

---

### 5. 自動化

1. 前往 **設定** → **自動化**。
2. 開啟 **「頁面載入後自動保存」**。
3. 設定 **延遲時間**（秒）— 頁面載入後等待指定秒數再自動保存。

## 🔒 隱私說明

Hiyori Tools **完全在本地端運行**。不會收集、追蹤或上傳任何資料至任何伺服器。Notion 同步功能僅使用你自己的 Token，直接與 Notion 官方 API 通訊。
