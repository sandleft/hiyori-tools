## 📖 概要

URLの一括管理やWebコンテンツのワンクリック収集ができる、軽量で効率的なブラウザ拡張機能です。日常のブックマーク、資料のアーカイブ、ナレッジマネジメントに最適です。

---

## ✨ 主な機能

### 🔗 URL管理
- URLを一括保存、ワンクリックで全タブを開く機能に対応
- 現在のウィンドウの全URLをワンクリックでコピー
- 除外リスト：指定したURLを自動でスキップ

### 📥 Web収集
- スクリーンショット保存（`.png`）
- ドキュメント保存（`.md`）
- ページソース保存（`.html`）
- 構造化データ保存（`.json`）
- PDF保存（ブラウザの印刷機能を呼び出し）
- Webページ全体スクリーンショット（縦長ページ自動結合）
- Notionへ同期（フィールド名カスタマイズ可能）

### ⚡ 自動化
- Webページを開いた後に自動保存、遅延時間の設定可能

### 🎀 特徴
- シンプルなUI、直感的な操作
- ローカルで動作、データをアップロードしません
- Notion API連携をサポート
- 不要なURLを自動除外

## 🚀 使い方

### 1. インストール

- Edge ブラウザのダウンロード先：https://microsoftedge.microsoft.com/addons/detail/hiyori-tools/ffkpgbobmimcehhcmehkadmkefblaidj
- Chrome のダウンロード先：審査中

インストール後、ブラウザのツールバーのパズルアイコンをクリックし、Hiyori Tools を**ピン留め**すると便利です。

---

### 2. URL管理

#### URLを一括保存・開く
1. Hiyori Tools のアイコンをクリックしてパネルを開きます。
2. テキストエリアにURLリストを貼り付けます（1行1URL）。
3. **「すべて開く」** をクリック — すべてのURLが新しいタブで一度に開きます。

#### 現在のウィンドウのURLを全コピー
1. パネルを開きます。
2. **「全URLをコピー」** をクリック — 現在のウィンドウのすべてのURLがクリップボードにコピーされます。

#### 除外リスト
1. **設定** → **除外リスト** を開きます。
2. 自動的にスキップしたいURLやドメインパターンを追加します。
3. 一括操作の際、これらのURLは無視されます。

---

### 3. Web収集

保存したいページを開いた状態で、Hiyori Tools のアイコンをクリックします。

| 形式 | 操作方法 |
|------|---------|
| 📸 スクリーンショット（`.png`） | **「スクリーンショットで保存」** をクリック — 表示エリアのスクリーンショットを保存 |
| 📄 ドキュメント（`.md`） | **「Markdownで保存」** をクリック — ページ内容を `.md` ファイルとして抽出 |
| 🌐 HTML（`.html`） | **「HTMLで保存」** をクリック — 現在のページソースを `.html` ファイルとして保存 |
| 🧩 JSON（`.json`） | **「JSONで保存」** をクリック — 現在のページの構造化データを `.json` ファイルとして保存 |
| 🖨️ PDF | **「PDFで保存」** をクリック — ブラウザの印刷ダイアログが開きます（「PDFに保存」を選択） |
| 📜 ページ全体のスクリーンショット | **「全体スクリーンショット」** をクリック — スクロール可能なページ全体をキャプチャして結合 |
| 🗂️ Notionへ同期 | 下記のNotion設定を参照 |

---

### 4. Notion連携の設定

1. [https://www.notion.so/my-integrations](https://www.notion.so/my-integrations) にアクセスし、新しいインテグレーションを作成します。
2. **インテグレーショントークン** をコピーします。
3. 対象のNotionデータベースを開き → **「...」** → **「コネクト」** → 作成したインテグレーションを接続します。
4. データベースのURL（最後の `/` と `?` の間の文字列）から **データベースID** をコピーします。
5. Hiyori Tools の **設定** → **Notion** を開きます：
   - **APIトークン** を貼り付け
   - **データベースID** を貼り付け
   - 必要に応じてフィールド名をカスタマイズ
6. 任意のWebページで **「Notionへ同期」** をクリックすると、データベースに保存されます。

---

### 5. 自動化

1. **設定** → **自動化** を開きます。
2. **「ページ読み込み後に自動保存」** をオンにします。
3. **遅延時間**（秒単位）を設定 — 指定した時間が経過してから自動保存が実行されます。

## 💡 利用シーン

- 毎日チェックするサイトを一括で開く
- 記事、チュートリアル、ニュースページのコレクション
- 重要なWebコンテンツの定期バックアップ
- WebページをNotionに同期して整理

---

## 🔒 プライバシー

Hiyori Tools は**完全にローカルで動作**します。データの収集・追跡・アップロードは一切行いません。Notion同期は、ご自身のトークンを使ってNotionの公式APIとのみ通信します。
