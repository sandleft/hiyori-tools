## 📖 소개

URL을 일괄 관리하고 웹 콘텐츠를 원클릭으로 수집할 수 있도록 돕는 가볍고 효율적인 브라우저 확장 프로그램입니다. 일상적인 북마크, 자료 보관, 지식 관리에 적합합니다.

---

## ✨ 주요 기능

### 🔗 URL 관리
- URL 일괄 저장, 원클릭으로 모든 탭 열기 지원
- 현재 창의 모든 URL 원클릭 복사
- 제외 목록: 지정된 URL 자동 건너뛰기

### 📥 웹 수집
- 스크린샷 저장 (`.png`)
- 문서 저장 (`.md`)
- PDF 저장 (브라우저 인쇄 호출)
- 전체 웹 페이지 스크린샷 (긴 페이지 자동 병합)
- Notion 동기화 (필드명 사용자 지정 가능)

### ⚡ 자동화
- 웹 페이지를 연 후 자동 저장, 지연 시간 설정 가능

### 🎀 특징
- 깔끔한 인터페이스, 직관적인 조작
- 로컬에서 실행, 데이터 업로드 안 함
- Notion API 통합 지원
- 원하지 않는 URL 자동 제외

## 🚀 사용 방법

### 1. 설치

[Chrome 웹 스토어](https://chromewebstore.google.com) 또는 [Microsoft Edge 추가 기능](https://microsoftedge.microsoft.com/addons)에서 **Hiyori Tools**를 설치하세요.

설치 후, 브라우저 툴바의 퍼즐 아이콘을 클릭하고 Hiyori Tools를 **고정**하면 편리하게 접근할 수 있습니다.

---

### 2. URL 관리

#### URL 일괄 저장 및 열기
1. Hiyori Tools 아이콘을 클릭하여 패널을 엽니다.
2. 텍스트 영역에 URL 목록을 붙여넣습니다 (한 줄에 하나씩).
3. **"모두 열기"** 클릭 — 모든 URL이 새 탭으로 한꺼번에 열립니다.

#### 현재 창의 URL 전체 복사
1. 패널을 엽니다.
2. **"전체 URL 복사"** 클릭 — 현재 창의 모든 URL이 클립보드에 복사됩니다.

#### 제외 목록
1. **설정** → **제외 목록**으로 이동합니다.
2. 자동으로 건너뛸 URL이나 도메인 패턴을 추가합니다.
3. 일괄 작업 시 해당 URL은 무시됩니다.

---

### 3. 웹 수집

저장하고 싶은 페이지로 이동한 후, Hiyori Tools 아이콘을 클릭합니다.

| 형식 | 사용 방법 |
|------|---------|
| 📸 스크린샷 (`.png`) | **"스크린샷으로 저장"** 클릭 — 현재 화면 영역 저장 |
| 📄 문서 (`.md`) | **"Markdown으로 저장"** 클릭 — 페이지 내용을 `.md` 파일로 추출 |
| 🖨️ PDF | **"PDF로 저장"** 클릭 — 브라우저 인쇄 대화상자 열림 ("PDF로 저장" 선택) |
| 📜 전체 페이지 스크린샷 | **"전체 페이지 스크린샷"** 클릭 — 전체 스크롤 페이지를 캡처하여 병합 |
| 🗂️ Notion 동기화 | 아래 Notion 설정 참조 |

---

### 4. Notion 연동 설정

1. [https://www.notion.so/my-integrations](https://www.notion.so/my-integrations)에서 새 통합을 만듭니다.
2. **내부 통합 토큰**을 복사합니다.
3. 대상 Notion 데이터베이스 열기 → **"..."** → **"연결"** → 통합 연결.
4. 데이터베이스 URL (마지막 `/`와 `?` 사이의 문자열)에서 **데이터베이스 ID**를 복사합니다.
5. Hiyori Tools에서 **설정** → **Notion**으로 이동:
   - **API 토큰** 붙여넣기
   - **데이터베이스 ID** 붙여넣기
   - 필요에 따라 필드명 사용자 지정
6. 원하는 웹 페이지에서 **"Notion에 동기화"** 클릭 시 데이터베이스에 저장됩니다.

---

### 5. 자동화

1. **설정** → **자동화**로 이동합니다.
2. **"페이지 로드 후 자동 저장"** 켜기.
3. **지연 시간** (초 단위) 설정 — 페이지 로드 후 지정한 시간이 지나면 자동으로 저장됩니다.

## 💡 활용 사례

- 매일 확인하는 웹사이트 일괄 열기
- 기사, 튜토리얼, 뉴스 페이지 북마크
- 중요한 웹 콘텐츠 정기 백업
- 웹 페이지를 Notion에 동기화하여 정리

## 🔒 개인 정보 보호

Hiyori Tools는 **완전히 로컬에서 실행**됩니다. 어떤 데이터도 수집, 추적 또는 서버에 업로드되지 않습니다. Notion 동기화는 사용자 본인의 토큰을 이용해 Notion 공식 API와만 통신합니다.
